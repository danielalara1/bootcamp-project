### Atajos del teclado

Utilizamos los atajos del teclado más frecuentes:

Al utilizar "Ctrl + I", usaremos el atajo principal para que la IA genere o edite nuestro código. Con "Ctrl + L" entramos al chat; aquí podemos realizar preguntas generales sobre el código y nos responderá de forma breve o específica. Por último, "Tab" se usa para aceptar las sugerencias automáticas de la IA.

Para comprobar estos atajos, utilicé el archivo "app.js" modificando algunas partes del código; esto permitió modificar funciones sin tener que borrar y escribir a mano.

### Refactorización

He realizado 5 refactorizaciones principales utilizando la IA.

Primero creamos una carpeta llamada "comprobaciones"; aquí movemos las validaciones. Cambié las variables raras por nombres que se entiendan y cada función hace una sola cosa. Finalmente para que la aplicación no falle al guardar datos añadimos "try... catch".
